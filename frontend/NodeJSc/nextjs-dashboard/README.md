## Next.js App 

# Creating a new nextJS app
npx create-next-app@latest

# Run app in dev mode
*(we run this command in **frontend** directory)* <br>
npm run dev

# Useful scripts
These scripts refer to the different stages of developing an application:

-dev: runs next dev to start Next.js in development mode. <br>
-build: runs next build to build the application for production usage. <br>
-start: runs next start to start a Next.js production server. <br>
-lint: runs next lint to set up Next.js' built-in ESLint configuration.
